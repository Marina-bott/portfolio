Vous avez un projet de développement d'Application ?

Je développe des applications mobile hybride
Avec la plateforme Angular qui est basé sur un langage TypeScript et son framework IONIC qui permet une personnalisation totale du design pour le côté client de l'application.

Application Mobile

DÉVELOPPEMENT HYBRIDE

Une application « cross-platform », qui bénéficie d'une conception plus rapide que les applications natives car seulement une version du code est nécessaire pour les différents systèmes d’exploitation (Apple, Android, Windows). Les développements en « cross-platform » ont l’avantage de réduire la maintenance et de faciliter l’ajout de fonctionnalités car il n’y a qu’une seule version du code. Une application « hybride » est une application ou le contenu web est « encapsulée » avec une interface mobile. On y retrouve alors par exemple du code HTML 5 comme pour les sites web. Pour ce faire,on utilise des SDK telles qu’Apache Cordova qui est le plus courant et également celui avec lequel je travail.

API REST

une API est une Interface dite Backend, elle permet notamment ( mais pas que!) la communication entre l'application mobile et le serveur de base de données. Elle permet aussi de coder une seule fois la logique métier et on peut ainsi y connecter autant d’interfaces utilisateurs que l'on veut (Intranet, application mobile, site web, etc).Toutes les API que je conçois sont réalisées en langage PHP. L'utilisation d'une API permet la sécurisation des échanges de données car en réalité, l'application mobile ne détient aucunes informations sensible, elle en reçoit juste une "copie" envoyée par l'API qui elle même est codée de manière à respecter toutes les normes de sécurité en vigueur et qui envoie les requêtes au serveur.

ANGULAR

Angular est un Framework open source écrit en JavaScript qui permet la création d’applications Web et mobile. Le framework s’appuie sur Typescript, une surcouche au JavaScript développé par Microsoft ! Il me permet donc par exemple de développer un site web et une application mobile en même temps sans avoir à gérer deux projets en parallèle. Pratique non ?



IONIC

Ionic est également un framework libre d'utilisation qui permet de développer des applications mobiles en utilisant les technologies et les langages du web comme HTML, CSS, JavaScript, Angular et TypeScript.Ionic est comme un kit de développement logiciel (SDK) pour la création d'applications mobiles hybrides. Il fournit une collection de composants qui imitent l'aspect, la convivialité et les fonctionnalités natives de chaque plateforme mobile. Tous ces composants sont des éléments que j'utilise dans la construction d'applications ioniques, ainsi que le plugins Cordova.
